# RDV-RECETAS-LANDING

Simple Vite + React prototype for the landing page.

Getting started

1. Install dependencies:

```
npm install
```

2. Run development server:

```
npm run dev
```

3. Build for production:

```
npm run build
```

Deploy

- This repo includes a GitHub Actions workflow to build and publish to GitHub Pages on push to `main`.
- Ensure `npm run build` produces the `dist/` folder (Vite default).
